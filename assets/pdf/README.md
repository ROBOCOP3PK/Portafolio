# CV PDF

**CV-David-Gonzalez.pdf**

## Cómo crear tu CV en PDF

Puedes crear tu CV en PDF de varias maneras:

1. **Desde Word/Google Docs**: Crea tu CV y exporta como PDF
2. **Canva**: Usa plantillas profesionales y descarga como PDF
3. **LaTeX**: Para CVs más técnicos y profesionales
4. **Herramientas online**: resume.io, novoresume.com, etc.

Asegúrate de que el archivo final se llame `CV-David-Gonzalez.pdf` y esté en esta carpeta.
